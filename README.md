## What is Random Spaces?

__Random Spaces__ is a free, open-source learning resources for students and researchers interested in applying Python to solve complex problems, to manage and tranform data, and to produce publication-quality figures. The tutorials here bridge the gap beetween condensed matter physics, chemistry and computational tools.
